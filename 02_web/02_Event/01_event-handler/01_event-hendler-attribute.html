01. 이벤트 핸들러 어트리뷰트 방식

이벤트 핸들러 등록

        이벤트 핸들러
        - 이벤트가 발생했을 때 브라우저에 호출을 위임한
        함수.
        이벤트가 발생했을 때 브라우저에게 이벤트 호출을
        위임하는 것을 이벤트 핸들러 등록이라고 하며,
        이벤트 핸들러를 등록하는 벙법은 3가지가 있다.

이벤트 핸들러 어트리뷰트

        HTML 이벤트 핸들러 어트리뷰트(on 접두사 + 이벤트 타입) 값으로
        함수 호출문을 할당하여 이벤트 핸들러를 등록하는 방식이다.
        주의점은 함수 참조가 아닌, 함수 호출문을 할당하는 것이다.
